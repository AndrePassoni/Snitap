## ğŸ’» Project

**Snitap** is a modern, responsive landing page concept for a rollerblade shop. The goal of this project was to practice layout structuring and advanced CSS styling techniques.

The interface highlights the skating lifestyle, featuring a dynamic hero section with animated text, an infinite scrolling banner, and a grid-based photo gallery.

## ğŸš€ Technologies

This project was developed with the following technologies:

- **HTML5** (Semantic structure)
- **CSS3** (Styling, Layout, and Animations)
- **Git & GitHub** (Version control)

## âœ¨ Features

- **Responsive Design**: Adapts to different screen sizes (Mobile First approach).
- **CSS Grid**: Used for the "Use Snitap" photo gallery layout.
- **Flexbox**: Used for alignment in the header, hero section, and footer.
- **CSS Animations**: 
  - `@keyframes` used for the sliding text effect in the Hero section.
  - Infinite rolling animation for the brand banner.
  - Smooth hover transitions on buttons and images.
- **CSS Variables**: Global color and font management for consistent theming.
